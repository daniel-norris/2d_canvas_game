# Arcade Game :video_game:

![Game](/atari_breakout.gif)

## About 

A copy of the Atari Breakout game released in 1976. Written using vanilla JS and the `canvas` element. 

## Install 

1. Clone the repo. 
```bash
git clone {url} {foldername}
```

2. Install `npm`. 
```bash
npm install
```

## Controls 

:arrow_backward: left   
:arrow_forward: right   
OR mouse enabled
